lopix interactive examples

This page links to a few interactive examples from lopix, a small package made to assist the development of simple Ebitengine pixel art games.

Examples have been slightly adapted and compiled to WASM so they can be executed directly on the web:

  • Color walk: an artsy but complete game in under 200 lines.
  • Filters: press left/right to see the behavior of different scaling filters.
  • Gametest: an example of the kind of game that you shouldn't be making with lopix. Provided as a reference to be able to compare with mipix.

You can find the sources at tinne26/lopix-examples/src.
